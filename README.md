# 📖 Dictionary App

A modern Dictionary App built with **React Native**, **Expo**, and **TypeScript**. The app allows users to search English words, view meanings, play pronunciation audio, save favorite words, and maintain search history.

## ✨ Features

- 🔍 Search any English word
- 📖 View word meanings and definitions
- 🔊 Listen to pronunciation audio
- ⭐ Save favorite words
- 🕘 Search history (stored locally)
- 📱 Bottom Tab Navigation
- 💾 Local storage using AsyncStorage
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack
